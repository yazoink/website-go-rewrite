Happy (late) new year!


About a month ago I decided to refurbish an old netbook to use as a portable laptop, since my ThinkPad T430 is very bulky and annoying to carry around.


A while ago, I was given a couple of broken old Acer Travelmate B110-E netbooks by a family member. They were originally used by a primary school, so you know they're only the finest quality.


They were both broken in different ways:


Laptop 1


  • Broken screen (lines all over)
  • Would not boot past BIOS splash screen or enter setup utility

Laptop 2


  • Screen/bezel hinge completely snapped and detached
  • Fan so rattly you can hear it from two rooms away
  • Sunken in trackpad that would not come back out

I was able to salvage a working laptop from the working parts (with a couple of upgrades and a thermal paste change) fairly easily.


The Hardware


The specs are really nothing special...

  • Intel Celeron 1007U processor (Ivybridge, 1.50 GHz, 2 cores/threads, 17W TDP)
  • 8 GB 1600 MT/s memory (upgraded from 3 GB of slower, mismatched memory)
  • 240 GB WD Green SATA SSD (upgraded from a 120 GB Crucial M500)
  • 1366x768 @ 60Hz LCD display

The only component I needed to purchase for this was a new battery which cost a total of $38.46 AUD from Aliexpress (with shipping). It's a pretty terrible battery (who would've guessed) but it works well enough that I'm satisfied.


It also has respectable IO:

  • x1 Ethernet
  • x1 VGA
  • x1 HDMI
  • x1 headphone/microphone jack
  • x3 USB 2.0
  • DC charging port (big fan of this)

As much as I wish it had USB 3.0 I would not expect a low-end machine from 2013 to have it, and it's not really a big deal. An SD card reader would be good as well, but also not a dealbreaker.


The form factor is very convenient, at 29.46 cm / 11.6" and weighing in at 1.276 kg / 2.8 lbs.




The Software


My OS of choice is NixOS, so that's what I installed on this thing. NixOS certainly isn't the most optimized Linux distribution I've tried (that award would easily go to Void) but the convenience of having all of my shared configurations in one nix flake is my primary reason for using it.


The window manager I ended up going with is SwayFX. I would usually use Hyprland but SwayFX uses marginally less resources (roughly 100 MB less RAM) so I went with it in order to get as light of a desktop as I can (with it still being adequately riceable and compatible with my workflow).


The other software I'm using is:

Kernel: Linux-zen
Web Browser: Firefox with Betterfox user.js and a mininal userChrome.css
Terminal: Foot
Shell: ZSH with Starship prompt
Status Bar: Waybar
App Launcher: Anyrun (considering switching if I can find another launcher that supports shadows)
Notification Daemon: SwayNC
Editor: Neovim
File Manager (GUI): Thunar
File Manager (TUI): Yazi
Display Manager: SDDM
Music Player: MPD/ncmpcpp

Performance


Web Browsing


I was honestly very surprised by how usable this machine is. Web browsing is generally very smooth, and it can comfortably play Youtube videos at up to 1080p with the help of enhanced-h264ify/Invidious.


The extensions I have installed are:


  • Ublock Origin
  • SponsorBlock
  • Stylus (rarely used on this machine)
  • ViolentMonkey (Also rarely used on this machine)
  • LocalCDN
  • LibRedirect
  • floccus bookmark sync
  • ClearURLs
  • Control Panel for Twitter
  • enhanced-h264ify
  • Pywalfox

Websites are slightly slower than they are on my laptop or PC (as expected) but it's not noticeable after a short adjustment period. I wouldn't describe it as "laggy", rather just "less smooth".


Discord


Vesktop with custom CSS and numerous plugins can get noticeably laggy but not to the point where it's unusable. The vanilla discord webapp runs fairly effortlessly in Firefox. I usually close Vesktop if I'm planning to do something resource intensive as it does tend to eat the CPU.


Gaming


I was not expecting much at all from this, but it definitely exceeded my expectations!


The first game I tried was Luanti/Minetest. It ranges from 20-60 FPS with the VoxeLibre mod, making it playable (but only just).


After that I installed Duke Nukem 3D with eDuke32 and it runs flawlessly. Same goes for Doom with GZDoom.


The next game I tried was Minecraft which definitely surprised me the most out of everything I tried...


On the latest version of Minecraft (1.21.4 currently) I was able to get it to run at an average of up to 78 FPS at an 80 FPS cap in a singleplayer world (with a few dips but still)!

Conclusion


TLDR; This 2013 netbook is usable in 2025, but only if you use a lightweight Linux setup and know how to optimize the software you're using.


Pros


  • Small and lightweight, making it very portable
  • Very durable
  • Repairable/modular design
  • Ethernet port
  • Distraction-free, due to less support for games
  • Very comfy
  • Quiet, even under load
  • Conversation starter, I guess

Cons


  • Relatively slow
  • Can't run many games
  • Shitty screen and awful speakers
  • Very short battery life and limited options for replacement batteries
  • No USB 3.0 or SD card reader
  • Tiny annoying arrow keys / overall tacky keyboard

So pretty much, I would not willingly use this as my main computer, but I reckon I could, hypothetically. I'm continuing to use this as a portable laptop for situations where I don't need anything better.


The moral of this story is that you don't need a new computer (unless you do).
